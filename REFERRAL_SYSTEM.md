# 🎁 Реферальная система AICEX One

## 📋 Обзор

Полноценная реферальная программа, позволяющая пользователям приглашать друзей и зарабатывать токены.

## 🎯 Как это работает

### **Для пригласившего:**
- Получает **+100 токенов** за каждого приглашенного друга
- Видит статистику всех своих рефералов
- Получает уведомление в боте когда друг присоединяется

### **Для приглашенного:**
- Получает **+50 токенов** бонусом при регистрации по реферальной ссылке
- Стандартные **100 токенов** стартовый бонус + **50 токенов** реферальный бонус = **150 токенов** всего

## 🔗 Реферальная ссылка

Формат: `https://t.me/YOUR_BOT?start=ref_TELEGRAM_ID`

Пример: `https://t.me/aicex_bot?start=ref_669231710`

## 💻 Техническая реализация

### **Backend:**

#### 1. **ReferralService** (`back/src/services/ReferralService.ts`)
- `processReferral()` - обрабатывает реферальную связь при регистрации
- `getReferralStats()` - возвращает статистику рефералов
- Автоматически начисляет бонусы обоим пользователям
- Записывает транзакции в `TokenHistory`

#### 2. **API Endpoints** (`back/src/routes/api.ts`)
```typescript
GET /api/referrals/stats     - Статистика рефералов
GET /api/referrals/list      - Список приглашенных друзей
```

#### 3. **Telegram Bot** (`back/src/bot/production-bot.ts`)
- Обрабатывает параметр `ref_TELEGRAM_ID` в команде `/start`
- Отправляет уведомления обоим пользователям
- Автоматически начисляет бонусы

### **Frontend:**

#### 1. **Страница "Пригласить друзей"** (`front/src/views/ReferralsView.vue`)
- Вкладка "Баланс" - реферальная ссылка и статистика
- Вкладка "Рефералы" - список приглашенных друзей
- Вкладка "Документация" - описание программы

#### 2. **Компонент Balance** (`front/src/views/ReferralsTabs/Balance.vue`)
- Реферальная ссылка с кнопкой **Copy**
- Статистика:
  - Приглашено друзей
  - Заработано токенов
  - Бонус за реферала
- История транзакций

## 📊 База данных

### **Поля в User:**
```typescript
referral: String?           // ObjectId пригласившего пользователя
friendsReferred: Int        // Счетчик приглашенных друзей
```

### **TokenHistory:**
```typescript
type: "REFERRAL"            // Бонус пригласившему
type: "BONUS"               // Бонус приглашенному
description: "Бонус за приглашение друга (@123456789)"
```

## 🧪 Как тестировать

### **1. Создание реферальной ссылки:**
1. Откройте мини-апп
2. Перейдите в "Пригласить друзей"
3. Скопируйте реферальную ссылку
4. Проверьте формат: `https://t.me/YOUR_BOT?start=ref_YOUR_TELEGRAM_ID`

### **2. Регистрация по реферальной ссылке:**
1. Откройте ссылку с другого аккаунта (или в incognito браузере)
2. Нажмите `/start` в боте
3. Проверьте уведомления:
   - Новый пользователь: "🎁 Вы зарегистрировались по реферальной ссылке! 💰 Вам начислено +50 токенов бонусом"
   - Пригласивший: "🎉 Ваш друг присоединился к AICEX One! 💰 Вам начислено +100 токенов"

### **3. Проверка статистики:**
1. Откройте мини-апп
2. Перейдите в "Пригласить друзей"
3. Проверьте обновление счетчиков:
   - Приглашено друзей: +1
   - Заработано токенов: +100

### **4. Проверка истории токенов:**
1. Откройте вкладку "Движение токенов"
2. Найдите транзакции типа "REFERRAL" и "BONUS"
3. Проверьте суммы: +100 и +50

## 🔒 Безопасность

- ✅ Нельзя пригласить самого себя
- ✅ Бонус начисляется только при первой регистрации
- ✅ Все транзакции логируются в `TokenHistory`
- ✅ Реферальная связь сохраняется в БД навсегда

## 📈 Метрики для отслеживания

1. **Конверсия рефералов** - % зарегистрировавшихся по ссылкам
2. **Активность рефералов** - сколько генераций делают приглашенные
3. **Топ пригласивших** - кто пригласил больше всего друзей
4. **Retention рефералов** - возвращаемость приглашенных пользователей
